About the Project

Hospital readmissions are a significant challenge in healthcare, leading to increased costs and reduced patient satisfaction. Our project leverages machine learning to predict the likelihood of diabetic patient readmissions, enabling proactive interventions and resource optimization.

Using advanced machine learning algorithms including Random Forests and XG Boost, LightGBM, we analyze patient data to identify key factors contributing to readmission risk.

Built using open‑source tools with the goal of improving patient outcomes and optimizing hospital resources.
